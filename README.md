# SENSO

Multi-Agent Organizational Intelligence Platform.

## Stack

- Vite + React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion
- React Router v6
- TanStack Query

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Tests

```sh
npm test
```
